## LLM

Large Language Models (LLMs) sind KI-Modelle, die aus sehr grossen Textmengen Muster lernen und dadurch natürliche Sprache verstehen sowie neue Texte erzeugen können. 

Sie werden genutzt, um Aufgaben wie Fragen beantworten, Texte Zusammenfassen, Übersetzen oder Codegenerierung effizient zu unterstützen.

Unterschied zu früher: LLMs sind universelle Modele statt aufgaben­spezifischer ML-Modelle.

### Hands-on

Installiert, lokal auf dem Rechner, Tools Eurer Wahl, z.B.
* Docker Desktop
* Ollama
* LMStudio (node.js)
* GPT4All (RAG integriert)

Versucht den gleichen Prompt, z.B. „wie baue ich einen Web Server in Python“ oder „wer war John F. Kennedy“ mit verschiedenen LLMs (gemma3, smollm2, …)

**ACHTUNG**: das braucht viel Speicherplatz auf Eurem Rechner!
